```
###### 6级标题
```

# 2. 字体格式
*斜体字      >>就是在字的前后各添加一个星号*  
_斜体字      >>也可以在字的前后各添加一个下划线_  
**粗体字     >>就是在字的前后各添加两个星号**  
__粗体字     >>也可以在字的前后各添加两个下划线__  
***粗斜体字  >>就是在字的前后各添加三个星号***  
___粗斜体字  >>也可以在字的前后各添加三个下划线___
